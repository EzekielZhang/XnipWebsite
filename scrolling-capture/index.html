How to use scrolling capture

GIF example of scrolling capture：


check the screenshot


1.Starting a capture

You can start a capture by shortcut or click the button in the menu bar icon.


2.Select the area

After start a capture, click and drag to select the capture area. A blue box will indicate the area. If you are making a scrolling capture, make sure you select the correct area.


3.Click the tool bar icon to start scrolling capture


4.Scroll the content

Once start a scrolling capture, use track pad or mouse to scroll the content you want to capture.
